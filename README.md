# Trilha Federal 🇧🇷

> O caminho para o ensino público de excelência

## 📋 Sobre o Projeto

O **Trilha Federal** é uma plataforma web desenvolvida para democratizar o acesso à informação sobre vestibulares de instituições públicas federais brasileiras. Nossa missão é tornar acessível informações que, apesar de serem públicas, são pouco conhecidas pela população.

### 🎯 Missão

Divulgar informações sobre vestibulares de universidades federais, institutos federais e escolas técnicas públicas que, apesar de serem públicas, são pouco conhecidas pela população.

### 💡 Visão

Um Brasil onde todos tenham conhecimento sobre as excelentes oportunidades de ensino público federal disponíveis, permitindo escolhas educacionais mais informadas.

### 🤝 Valores

- **Transparência**: Informações claras e acessíveis
- **Democratização**: Acesso igualitário à informação educacional
- **Excelência**: Compromisso com a qualidade da educação pública
- **Desenvolvimento Social**: Transformação através da educação

## 🚀 Como Executar

1. **Clone o repositório**:

   ```bash
   git clone [url-do-repositorio]
   cd trilhafederal
   ```

2. **Abra o arquivo principal**:

   - Abra `index.html` em seu navegador
   - Ou use um servidor local:
     ```bash
     python -m http.server 8000
     # Acesse http://localhost:8000
     ```

3. **Estrutura de arquivos**:
   ```
   trilhafederal/
   ├── index.html              # Página principal
   ├── internacional.html      # Trilha Internacional
   ├── equipe.html            # Nossa Equipe
   ├── styles.css             # Estilos CSS
   ├── script.js              # JavaScript
   ├── assets/                # Imagens e recursos
   │   ├── logo.png
   │   ├── logo-cortada.ico
   │   └── trilha federal sem fundo.png
   └── README.md              # Documentação
   ```
