# 🚀 Trilha Federal

> O caminho para o ensino público de excelência começa aqui.

## 📋 Sobre o Projeto

O **Trilha Federal** é uma plataforma dedicada a democratizar o acesso à informação sobre vestibulares de universidades federais, institutos federais, escolas técnicas e outras instituições públicas de ensino superior no Brasil.

### 🎯 Missão

Divulgar informações sobre vestibulares de instituições públicas federais que, apesar de serem públicas, são pouco conhecidas pela população, permitindo que mais estudantes tenham acesso a oportunidades de ensino de qualidade.

### 💡 Visão

Um Brasil onde todos tenham conhecimento sobre as excelentes oportunidades de ensino público federal disponíveis, permitindo escolhas educacionais mais informadas.

### 🤝 Valores

- **Transparência**: Informações claras e acessíveis
- **Democratização**: Acesso universal à informação educacional
- **Excelência**: Foco na qualidade do ensino público
- **Compromisso social**: Desenvolvimento através da educação

## 🛠️ Como Executar

1. **Clone o repositório**

   ```bash
   git clone https://github.com/seu-usuario/trilhafederal.git
   cd trilhafederal
   ```

2. **Abra o arquivo index.html**

   - Use um servidor local (recomendado)
   - Ou abra diretamente no navegador

3. **Servidor local (opcional)**

   ```bash
   # Python 3
   python -m http.server 8000

   # Node.js (com http-server)
   npx http-server

   # PHP
   php -S localhost:8000
   ```

## 🎨 Personalização

### Cores

As cores podem ser facilmente alteradas editando as variáveis CSS no arquivo `styles.css`:

```css
:root {
  --primary-green: #009c3b;
  --primary-yellow: #ffdf00;
  --primary-blue: #002776;
  /* ... outras cores */
}
```

### Conteúdo

- Edite o arquivo `index.html` para alterar textos e estrutura
- Modifique `styles.css` para ajustar estilos
- Atualize `script.js` para funcionalidades JavaScript
